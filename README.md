# Command-line Stock Checker

Tool to display price information for a given ticker on the command line.

## Installation

- Install Rust: https://rust-lang.org/tools/install/
- Install the tool:

```sh
git clone https://github.com/societymartingale/stock-checker.git
cd stock-checker
cargo build --release
./target/release/vol -h
```

## Usage

```sh
vol -h
Usage: vol [OPTIONS] --ticker <TICKER>

Options:
  -t, --ticker <TICKER>  ticker symbol such as MSFT
  -r, --range <RANGE>    historical time range [default: m1] [possible values: d1, d5, m1, m3, m6, y1, y2, y5, y10, ytd, max]
  -h, --help             Print help
```
